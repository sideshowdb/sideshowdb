# Playground

Use the playground to inspect a real public repository, review its refs, and
compare the fetched GitHub view with the Sideshowdb interpretation panel.

The first release is intentionally limited:

1. Public GitHub repositories only.
2. Read-only exploration only.
3. Sample-repo fallback when a repository is missing, unsupported, or rate-limited.

Recommended path:

1. Start with `sideshowdb/sideshowdb`.
2. Use the sample link to see the happy path first.
3. Enter your own `owner/repo` pair once you understand the flow.
