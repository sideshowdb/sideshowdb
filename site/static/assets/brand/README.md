# SideshowDB Brand Assets

This folder contains exploratory SideshowDB carousel brand assets for docs-site,
GitHub organization, and repository usage.

## Raster

- `concepts/sideshowdb-carousel-concepts-v1.png` — first four-direction concept board.
- `concepts/sideshowdb-carousel-refinements-v1.png` — refined concept board for the selected directions.
- `raster/sideshowdb-carousel-refinements-strip.png` — horizontal strip of the four refined marks.
- `raster/*-logo.png` — raster logo lockups with the SideshowDB wordmark.
- `raster/*.png` without `-logo` — individual raster icon crops from the refined board.

## SVG

- `svg/*-icon.svg` — hand-redrawn vector icon interpretations.
- `svg/*-logo.svg` — vector logo lockups with the SideshowDB wordmark.
- `svg/sideshowdb-carousel-refinements-strip.svg` — vector strip for a branding page preview.

## Palette

- Deep ink: `#07324a`
- Carousel teal: `#009c98`
- Coral: `#ff4b45`
- Gold: `#ffb000`
- Sky blue: `#2e8deb`
- Paper: `#fff9ec`
